## 软件工程

### 阿里巴巴发布 Qoder 1.0，迈向智能体开发

**事件：** 阿里巴巴发布了其AI编程工具的升级版Qoder 1.0，产品定位从AI辅助的集成开发环境（IDE）升级为智能体自主开发工作台。用户只需定义需求，Agent团队便可自主完成后续全流程任务。

**解读：** 这代表了AI赋能软件开发范式的又一次跃迁，从辅助人类编写代码（Copilot模式），进化到接管部分开发流程（Agent模式）。这改变了对未来软件工程效率和开发者角色的判断，低代码/无代码的边界被进一步拓宽，软件开发的成本结构有望被重塑。

### GitHub 测试通用无障碍修复 AI Agent

**事件：** GitHub正在测试一个通用的无障碍功能（accessibility）AI代理，它不仅能在Copilot中回答相关问题，还能自动修复前端代码中常见的简单无障碍缺陷。在测试中，它已审查3535个拉取请求（PR），问题解决率达到68%。

**解读：** 这标志着AI在软件开发中的角色，正从“代码生成建议”向“代码质量保障和自动修复”演进。它改变了对未来软件开发工作流的判断，特定领域的代码审查和维护工作将被自动化，从而改变软件工程的成本结构和效率边界。

评分 73 · 来源 [The GitHub Blog - Natalie Guevara](https://github.blog/ai-and-ml/github-copilot/building-a-general-purpose-accessibility-agent-and-what-we-learned-in-the-process/)
