## 软件工程

### GitHub Copilot CLI 推出交互与非交互双模式

**事件：** GitHub Copilot CLI 正式区分两种调用模式：交互模式下用户输入 `copilot` 进入类聊天会话，支持通过 `/resume` 恢复历史上下文；非交互模式使用 `copilot -p "prompt"` 直接返回单次结果，适用于 shell 脚本集成与一次性代码生成任务。

**解读：** 该设计将 AI 辅助编程的系统入口从“单次问答”扩展为“可保持状态的终端会话”，允许开发者在 shell 环境中维持跨命令的连续上下文，降低长流程开发中的认知切换与重复 prompt 成本。

评分 58 · 来源 [The GitHub Blog - Kayla Cinnamon](https://github.blog/ai-and-ml/github-copilot/github-copilot-cli-for-beginners-interactive-v-non-interactive-mode/)
