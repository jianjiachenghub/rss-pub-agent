**播客脚本：个人日报**

**主持人 A：** 大家好，欢迎收听今天的个人日报。

**主持人 B：** 你好。

**主持人 A：** 今天的内容有个很有意思的对比：一边是 AI 正在大刀阔斧地重塑职场和开发者的工作流，甚至开始“动刀”管理层；另一边，全球资金却在不确定的环境里寻找确定的避风港。我们先从那个最值得关注的变化说起。

**主持人 B：** 确实，AI 对职场的渗透比我们想象的要深。瑞典隆德大学刚研发出一个新 AI 模型，只靠一次血液检测，就能识别多种神经退行性疾病。

**主持人 A：** 这对早期筛查太重要了。以前这类病确诊往往很晚，如果抽个血就能辅助诊断，等于把防线前移了一大步。这不仅是技术突破，更是给临床医生提供了一个全新的“眼睛”。

**主持人 B：** 没错。而在开发领域，AI 更是在成为基础设施。LM Studio 新版支持本地运行 Gemma 4，还能直接给 Claude Code 当后端。

**主持人 A：** 这其实改变了开发者的成本逻辑。以前用 API 是按次计费，现在转为了本地算力折旧。加上那个开源的 Nanocode 项目，复现顶尖代码模型的门槛在降低，看来“私有化部署”正变成主流选项。

**主持人 B：** 但能力越强，风险越大。最近有程序员专门做了个开源工具，用来拦截 AI 智能体的实时操作。起因是他发现像 Claude Code 这种工具有完整权限，但他根本不知道 AI 下一步会干啥。

**主持人 A：** 这确实是个隐患。现在的 Agent 权限太大了，就像给了个实习生管理员密码，必须得有个“安全带”。这也解释了为什么 GitHub Codex 改了定价，全面转向按 Token 收费——高频复杂任务的边际成本在上升，逼着大家更精细地管理权限和消耗。

**主持人 B：** 这种精细化管理的趋势甚至延伸到了企业组织架构。Block 创始人 Jack Dorsey 提出，AI 正在“杀死”中层管理。

**主持人 A：** 他裁员了 4000 人，理由很直接：中层管理的核心职能是信息传递和协调，现在 AI 接管了这部分。企业变得更扁平、敏捷是好事，但对在这个岗位上的人来说，这可能是最残酷的职场警钟。

**主持人 B：** 说完职场风险，我们看看全球资金的流向。香港财政司司长陈茂波透露，今年首季 IPO 集资额全球第一，排队申请超过 500 宗。

**主持人 A：** 这数据和欧美市场的动荡反差太大了。英国房地产正因为滞胀担忧遭遇信心危机，印度银行股也蒸发了 950 亿美元。在外围环境不明朗时，更多企业把香港视作融资和出海的重要窗口，资金的避险情绪很明显。

**主持人 B：** 这种避险逻辑在日本体现为另一种方式：机器人。日本正在加速部署物理 AI，但不是为了替代人工，而是填补没人干的岗位。

**主持人 A：** 对，因为劳动力短缺，机器人接管的都是低需求岗位。这个逻辑很务实，从“替代人工”变成了“填补空缺”，给老龄化社会的自动化打了个样。

**主持人 B：** 最后聊聊商业。投资人段永平改口了，实地考察后把泡泡玛特封为中国出海标杆。

**主持人 A：** 他以前说看不懂，现在发现店里全是外国人。这意味着市场对它的估值逻辑变了：从“看不懂的玩具公司”变成了“可理解的消费品牌”。这和茅台涨价是一个道理——都在不确定的环境里，验证了自己核心的定价权，让利润实实在在地回流。

**主持人 B：** 所以你看，无论是 AI 重塑职场，还是资金寻找避风港，核心都在于“确定性”。掌握了定价权或者技术主导权，才能在波动里站稳脚跟。

**主持人 A：** 好的，以上就是今天个人日报的主要内容，我们下期见。
