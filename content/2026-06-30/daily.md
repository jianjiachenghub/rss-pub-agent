## 软件工程

### Google 开源面向 Agent 开发的 Cloud CLI

**事件：** Google 开源了 `agents-cli`，这是一个 Python 命令行工具，可让编码助手在 Google Cloud 上创建、评估并部署 AI Agent；该项目目前在 GitHub 获得 3,780 个 star 和 436 次 fork。

**解读：** 开发者工具的系统入口正在从 IDE 插件扩展到云端 Agent 生命周期管理。评估和部署被放进同一个 CLI 后，团队更容易把 Agent 工作负载绑定到 Google Cloud 的运行、评测和发布链路上。

评分 95 · 来源 [Trending repositories on GitHub today · GitHub - google](https://github.com/google/agents-cli)

### Meta 开源面向 AI Agent 的设计系统 Astryx

**事件：** Meta 旗下 GitHub 账号开源了 `astryx`，这是一个用 TypeScript 编写、可定制的 AI Agent 设计系统，目前获得 1,378 个 star 和 82 次 fork。

**解读：** Agent 产品形态正在从单一聊天框转向可复用的界面组件。设计系统会改变企业构建 Agent 应用的系统入口和成本结构，尤其是把会话、任务状态、工具调用结果等交互模式沉淀为标准组件。
