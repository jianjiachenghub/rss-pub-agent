【播客脚本：个人日报】

**主持人 A**：各位好，欢迎收听今天的个人日报，我是 A。
**主持人 B**：我是 B。今天的内容真的让人感觉节奏很快，不管是大厂的人事变动，还是职场里的新状况，都值得咱们细聊一下。

**主持人 A**：没错。如果要划个重点，我觉得今天的主线是“边界与博弈”。巨头在抢占 AI 硬件的坑位，模型厂商在堵“薅羊毛”的漏洞，而普通打工人在面对“数字分身”带来的伦理冲击。

**主持人 B**：确实，这种博弈感特别强。咱们先看大厂那边，Meta 有了新动作。原小米和字节的硬件高管徐睿，确认加入 Meta 了，他要领导的是新组建的 AI 硬件团队。

**主持人 A**：这个履历很亮眼啊，徐睿之前在小米参与过转型，在字节也负责过硬件研发。Meta 这波操作，明显是想在 AI 硬件这个新赛道上加速，毕竟接下来不仅是拼模型，还得拼落地设备，这步棋走得很稳。

**主持人 B**：硬件要抢人，商业上也是强买强卖。马斯克最近要求参与 SpaceX IPO 的投行，必须购买他旗下 Grok 的订阅服务。

**主持人 A**：这招真是够硬核的。把 IPO 承销资格和 AI 服务绑定，相当于利用资本市场的地位强行给自家产品导流。这对投行来说，服务条款变贵了，但也侧面说明，AI 产品的分发竞争已经延伸到了这种非技术领域。

**主持人 B**：说回模型圈，阿里这几天风头很劲。刚发布的千问 3.6-Plus，才一天就冲上了 OpenRouter 的榜首，调用量破了纪录，被称为“中国编程能力最强模型”。

**主持人 A**：关键是价格还比 Claude API 低，这种高性价比加上超长上下文，对开发者很有吸引力。而且不仅是软件层，DeepSeek V4 传出消息，优先适配华为、寒武纪等国产芯片，大厂都在提前抢货。

**主持人 B**：这说明国产算力真的开始进入主流供应链了。以后咱们聊算力，可能真的不再只盯着英伟达了，这个格局的变化比单纯的技术迭代影响更深远。

**主持人 A**：技术进步快，矛盾也来得快。Claude 最近搞了个大动作，正式宣布封杀 OpenClaw 等第三方平台，理由是第三方“薅羊毛”导致系统过载，订阅费只能用于官方渠道。

**主持人 B**：对，这也引发了退订潮。开发者这边挺纠结的，一边是官方封杀第三方，一边是 Claude Code 又送了 100 美元额度来安抚用户。看来 Anthropic 是想把这波流量死死按在自己手里，不想被中间商赚差价。

**主持人 A**：这对咱们普通用户也有影响。最近大家都在讨论 Coding Plan 和 GPT Plus 怎么选。其实结论很务实：如果不考虑合规问题，国产 Coding Plan 便宜大碗，适合做日常辅助；真要搞定复杂任务，GPT Plus 或者官方 API 还是更稳一点。

**主持人 B**：工具选好了，职场环境却变了。今天有个帖子传得很火，说同事上午被优化，下午公司就把他的文档、聊天记录喂给 AI，造了个“数字分身”继续回答问题。

**主持人 A**：这听着挺让人后背发凉的。人走了，经验还得留下“当牛做马”。这种把员工知识资产直接剥离的做法，已经在圈里引发了关于隐私和伦理的激烈讨论，甚至有人开发反制工具来对抗这种“数字剥削”。

**主持人 B**：最后看一眼投资圈。段永平亲自去调研了泡泡玛特，六天之内从质疑转向认可，觉得符合“好生意”的标准。

**主持人 A**：这种一线投资者的转向，其实是在重新定义消费品牌的估值逻辑——跨文化的穿透力，可能比本土占有率更值钱。咱们做产品、看趋势，确实得把视野打开一点。

**主持人 B**：是的。今天的日报就先聊到这，AI 在重塑行业，也在重塑我们的职场和认知。

**主持人 A**：咱们下期见。
